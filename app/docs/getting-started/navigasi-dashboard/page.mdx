# Navigasi Dashboard

Dashboard adalah pusat kendali utama di Daily Worker Hub. Dari sini, Anda dapat mengakses semua fitur platform — mulai dari pencarian lowongan, pengelolaan lamaran, chat dengan pemberi kerja, hingga pencairan dana. Halaman ini menjelaskan setiap bagian dashboard secara detail.

---

## Tampilan Dashboard Pekerja

Jika Anda mendaftar sebagai **pekerja**, berikut adalah tampilan dan fungsi setiap bagian dashboard:

### Struktur Menu Pekerja

```
┌──────────────────────────────────────────────────────────────────┐
│                    DASHBOARD PEKERJA                               │
│  [Logo]  Cari Lowongan  |  Lamaran Saya  |  Saldo  |  Profil    │
└──────────────────────────────────────────────────────────────────┘
```

### Menu-Menu Utama

| Menu | Fungsi |Shortcut |
|------|--------|---------|
| **Beranda** | Overview: lowongan terbaru, notifikasi, statistik | Klik logo Daily Worker Hub |
| **Cari Lowongan** | Pencarian dan filter lowongan | — |
| **Lamaran Saya** | Daftar lamaran yang sudah apply dan statusnya | — |
| **Chat** | Percakapan dengan pemberi kerja | — |
| **Saldo Saya** | Saldo platform, riwayat transaksi, tarik dana | — |
| **Profil** | Edit profil, verifikasi, pengaturan akun | — |

### Beranda (Home Dashboard Pekerja)

**Konten yang ditampilkan:**

| Bagian | Deskripsi |
|--------|-----------|
| **Notifikasi Terbaru** | Alert lamaran diterima, pesan dari pemberi kerja, update platform |
| **Lowongan Sesuai Preferensi** | Rekomendasi lowongan berdasarkan profil dan lokasi |
| **Lowongan Terbaru** | Daftar 10 lowongan terbaru di platform |
| **Statistik Saya** | Total lamaran, lamaran aktif, total earning bulan ini |
| **Banner Promo** | Info fitur premium, event, atau update platform |

### Menu Cari Lowongan

**Fitur yang tersedia:**

| Fitur | Deskripsi |
|-------|-----------|
| **Search Bar** | Ketik kata kunci untuk pencarian cepat |
| **Filter Lokasi** | Pilih area: Seminyak, Kuta, Canggu, Ubud, dll |
| **Filter Kategori** | Pilih jenis pekerjaan: Waiter, Kitchen, Housekeeping, dll |
| **Filter Rate** | Masukkan range rate harian minimum dan maksimum |
| **Filter Jadwal** | Pilih hari kerja: weekdays, weekends, tanggal spesifik |
| **Sortir** | Urutkan: Terbaru, Terdekat, Bayaran Tertinggi |
| **Bookmark** | Simpan lowongan menarik untuk apply nanti |
| **Apply** | Lamar langsung dengan satu klik |

### Menu Lamaran Saya

**Struktur halaman:**

| Tab | Menampilkan |
|-----|-------------|
| **Semua** | Semua lamaran yang pernah Anda apply |
| **Pending** | Lamaran yang belum mendapat respons dari pemberi kerja |
| **Accepted** | Lamaran yang diterima — lanjut ke tahap hiring |
| **Rejected** | Lamaran yang ditolak pemberi kerja |
| **Expired** | Lowongan sudah ditutup sebelum Anda diterima |

**Detail setiap lamaran:**

| Informasi | Deskripsi |
|----------|-----------|
| **Nama Pemberi Kerja** | Bisnis yang memposting lowongan |
| **Posisi** | Jabatan yang Anda lamar |
| **Lokasi** | Area tempat kerja |
| **Rate** | Bayaran harian yang ditawarkan |
| **Status** | Pending / Accepted / Rejected / Expired |
| **Tanggal Apply** | Kapan Anda mengajukan lamaran |
| **Aksi** | Lihat detail, chat pemberi kerja, cancel lamaran |

### Menu Saldo Saya

**Struktur halaman:**

| Bagian | Deskripsi |
|--------|-----------|
| **Saldo Available** | Jumlah yang dapat diwithdraw saat ini |
| **Saldo Pending** | Dana yang sedang dalam proses (belum dirilis) |
| **Total Earning** | Semuaearning sejak awal bergabung |
| **Earning Bulan Ini** | Total earning pada bulan berjalan |
| **Tombol Tarik Dana** | Aksi untuk cairkan saldo ke rekening |

**Riwayat Transaksi:**

| Kolom | Deskripsi |
|-------|-----------|
| **Tanggal** | Kapan transaksi terjadi |
| **Jenis** | Rilis dari Dana Jaminan / Tarik Dana / Bonus |
| **Jumlah** | Nominal transaksi |
| **Dari** | Pemberi Kerja yang membayarkan (untuk release) |
| **Status** | Completed / Pending / Failed |

### Menu Profil

**Sub-menu yang tersedia:**

| Sub-menu | Fungsi |
|----------|--------|
| **Edit Profil** | Ubah foto, nama, deskripsi, keahlian |
| **Verifikasi Akun** | Lihat status verifikasi, lengkapi yang belum |
| **Portofolio** | Upload foto/video hasil kerja (opsional) |
| **Rating & Review** | Lihat review dari pemberi kerja |
| **Pengaturan** | Ubah password, aktifkan 2FA, atur notifikasi |
| **Keluar** | Logout dari akun |

## Tampilan Dashboard Pemberi Kerja

Jika Anda mendaftar sebagai **pemberi kerja**, berikut adalah tampilan dan fungsi setiap bagian dashboard:

### Struktur Menu Pemberi Kerja

```
┌──────────────────────────────────────────────────────────────────┐
│                 DASHBOARD PEMBERI KERJA                            │
│  [Logo]  Post Lowongan  |  Lowongan Saya  |  Saldo  |  Profil   │
└──────────────────────────────────────────────────────────────────┘
```

### Menu-Menu Utama

| Menu | Fungsi |
|------|--------|
| **Beranda** | Overview: lowongan aktif, pelamar terbaru, statistik hiring |
| **Post Lowongan** | Buat dan publikasikan lowongan baru |
| **Lowongan Saya** | Kelola semua lowongan yang diposting |
| **Database Pekerja** | Cari dan lihat profil pekerja yang terdaftar |
| **Chat** | Percakapan dengan pelamar |
| **Saldo** | Deposit, lihat riwayat transaksi Dana Jaminan |
| **Profil** | Edit profil bisnis, verifikasi, pengaturan |

### Beranda (Home Dashboard Pemberi Kerja)

**Konten yang ditampilkan:**

| Bagian | Deskripsi |
|--------|-----------|
| **Notifikasi Terbaru** | Alert pelamar baru, pesan dari pekerja, update platform |
| **Lowongan Aktif** | Jumlah lowongan yang sedang open |
| **Pelamar Baru** | Daftar pekerja yang baru melamar hari ini |
| **Transaksi Pending** | Hiring yang sedang dalam proses |
| **Statistik Hiring** | Total pekerja hired, total spending bulan ini |
| **Quick Actions** | Tombol cepat: Post Lowongan, Deposit Saldo |

### Menu Post Lowongan

**Formulir yang perlu diisi:**

| Field | Wajib? | Deskripsi |
|-------|--------|-----------|
| **Judul** | ✅ | Posisi yang dibutuhkan |
| **Kategori** | ✅ | Jenis pekerjaan |
| **Lokasi** | ✅ | Alamat atau area |
| **Tanggal** | ✅ | Tanggal dan jam kerja |
| **Rate** | ✅ | Bayaran harian |
| **Deskripsi** | ✅ | Detail tugas dan persyaratan |
| **Foto** | Opsional | Gambar tempat kerja |
| **Persyaratan** | Opsional | Sertifikat, pengalaman, dll |

### Menu Lowongan Saya

**Tab pada halaman:**

| Tab | Menampilkan |
|-----|-------------|
| **Semua** | Semua lowongan yang pernah diposting |
| **Aktif** | Lowongan yang sedang open |
| **Draft** | Lowongan yang belum dipublish |
| **Hired** | Hiring yang sedang aktif |
| **Completed** | Lowongan yang sudah selesai |

**Aksi per lowongan:**

| Aksi | Fungsi |
|------|--------|
| **Lihat Pelamar** | Lihat dan seleksi pekerja yang melamar |
| **Edit** | Ubah detail lowongan (jika belum ada yang hired) |
| **Tutup** | Tutup lowongan lebih awal |
| **Repost** | Buat duplikat untuk posting ulang |
| **Hapus** | Hapus lowongan draft |

### Menu Saldo

**Sub-menu:**

| Sub-menu | Fungsi |
|----------|--------|
| **Ringkasan Saldo** | Total saldo Dana Jaminan yang tersedia |
| **Deposit** | Isi saldo Dana Jaminan via bank transfer / e-wallet |
| **Riwayat Deposit** | Log semua deposit yang pernah dilakukan |
| **Transaksi Aktif** | Hiring yang sedang berlangsung |
| **Riwayat Transaksi** | Semua transaksi Dana Jaminan yang sudah selesai |

## Elemen UI yang Penting

### Ikon dan Indikator

| Ikon | Artinya |
|------|---------|
| 🔔 | Notifikasi — ada pesan atau update baru |
| ✅ | Terselesaikan / Lulus / Approved |
| ❌ | Ditolak / Gagal / Cancelled |
| ⏳ | Pending / Sedang diproses |
| ⭐ | Rating / Review |
| 💰 | Saldo / Transaksi finansial |
| 📍 | Lokasi |
| 📅 | Tanggal / Jadwal |
| ⭐ (badge) | Pekerja Premium / Terverifikasi |

### Status Lamaran dan Lowongan

| Status | Warna | Arti |
|--------|-------|------|
| **Aktif / Open** | Hijau | Lowongan masih menerima pelamar |
| **Pending** | Kuning | Menunggu respons dari pihak lain |
| **Accepted / Hired** | Biru | Lamaran diterima, proses hiring dimulai |
| **Rejected** | Merah | Ditolak atau tidak sesuai |
| **Completed** | Abu-abu | Selesai dieksekusi |
| **Expired / Closed** | Abu-abu | Sudah tidak aktif |
| **Cancelled** | Merah | Dibatalkan oleh salah satu pihak |
| **Diperseilisihkan** | Oranye | Ada perselisihan yang sedang diproses |

## Tips Navigasi

| Tips | Penjelasan |
|------|------------|
| **Gunakan search** | Untuk menemukan pekerja atau lowongan dengan cepat |
| **Aktifkan notifikasi push** | Agar tidak ketinggalan lamaran atau pelamar baru |
| **Filter sesuai kebutuhan** | Narrow down hasil pencarian untuk hasil yang lebih relevan |
| **Bookmark lowongan menarik** | Untuk apply nanti tanpa harus cari ulang |
| **Cek dashboard rutin** | Terutama pagi hari untuk lowongan hari yang sama |
| **Update profil secara berkala** | Meningkatkan visibilitas dan peluang diterima |

---

## Troubleshooting Navigasi

| Masalah | Solusi |
|---------|--------|
| **Menu tidak muncul** | Refresh halaman atau clear cache browser |
| **Error saat load data** | Cek koneksi internet, coba lagi dalam beberapa menit |
| **Notifikasi tidak masuk** | Aktifkan notifikasi di pengaturan browser/perangkat |
| **Halaman blank** | Logout dan login kembali, atau gunakan browser lain |
| **Data tidak update** | Pull-to-refresh di mobile, atau klik refresh di desktop |
