# Getting Started

Panduan awal untuk memulai perjalanan Anda sebagai daily worker di DailyWorkerHub.

## Apa itu DailyWorkerHub?

DailyWorkerHub adalah platform yang menghubungkan daily workers dengan pemberi kerja untuk kerja harian. Platform kami menyediakan sistem proteksi seperti Protection Pool dan Escrow System untuk memastikan keamanan transaksi.
