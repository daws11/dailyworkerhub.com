# Escrow System

Sistem pembayaran escrow yang aman untuk menjamin dana sampai pekerjaan selesai.

## Apa itu Escrow?

Escrow adalah layanan pihak ketiga yang menahan dana pembayaran sampai kondisi tertentu terpenuhi. Dengan cara ini, baik employer maupun worker memiliki jaminan keamanan.

## Alur Escrow System

### 1. Employer Deposit Dana
Employer deposit dana ke escrow system sebelum pekerjaan dimulai. Dana dihold oleh platform.

### 2. Worker Menyelesaikan Pekerjaan
Worker melakukan pekerjaan sesuai agreement. Setelah selesai, worker submit completion notice.

### 3. Employer Verifikasi dan Approve
Employer memverifikasi bahwa pekerjaan sudah sesuai. Jika sesuai, employer approve release dana.

### 4. Dana Released ke Worker
Setelah approval, dana released dari escrow ke worker. Worker dapat withdraw dana ke rekening mereka.

## Keuntungan Escrow System

### Untuk Employer
- Dana tidak hilang jika pekerjaan tidak selesai
- Leverage untuk memastikan worker menyelesaikan work
- Refund tersedia jika terjadi dispute

### Untuk Worker
- Jaminan pembayaran jika menyelesaikan pekerjaan
- Tidak perlu chase payment setelah work selesai
- Dana aman dalam escrow sampai approval

## Biaya Escrow

Terdapat small transaction fee untuk penggunaan escrow:
- Employer menanggung small fee saat deposit
- Worker menerima net amount setelah fee deduction

## Waktu Proses

- **Deposit**: Segera setelah pembayaran berhasil
- **Release**: 1-3 hari kerja setelah approval dari employer
- **Dispute Hold**: Dana tetap dihold sampai dispute resolved

## Keamanan

Escrow system DailyWorkerHub menyediakan:
- Dana terpisah dari operational funds
- Enkripsi data transaksi
- Two-factor authorization untuk release
- Audit trail lengkap untuk setiap transaksi
