# Fitur Platform Daily Worker Hub

Daily Worker Hub dibangun dengan fondasi keamanan dan kepercayaan untuk ekosistem tenaga kerja harian di Bali. Berikut adalah fitur-fitur utama yang membedakan kami dari platform lainnya dan melindungi kedua belah pihak dalam setiap transaksi.

---

## Gambaran Umum Fitur

| Fitur | Kategori | Kategori | Benefit Utama |
|-------|----------|----------|---------------|
| **Sistem Dana Jaminan** | Keamanan Pembayaran | Premium | Dana gaji diholdpihak ketiga sampai pekerjaan selesai |
| **Dana Perlindungan** | Keamanan Transaksi | Premium | Dana proteksi untuk menangani perselisihan dan penipuan |
| **Cari Lowongan** | Pencarian | Free | Filter advanced berdasarkan lokasi, jenis, rate, jadwal |
| **Posting Lowongan** | Penawaran | Free | Posting dengan deskripsi lengkap dan persyaratan jelas |
| **Chat Real-Time** | Komunikasi | Free | Komunikasi langsung pekerja dan pemberi kerja |
| **Rating & Review** | Reputasi | Free | Build reputation untuk kedua belah pihak |
| **Top-up Saldo** | Pembayaran | Free | Pemberi Kerja deposit untuk Dana Jaminan |
| **Tarik Dana** | Pencairan | Free | Pekerja cairkan dana ke rekening bank |

## Fitur Keamanan Utama

### Sistem Dana Jaminan

Sistem pembayaran Dana Jaminan yang menahan dana gaji dipihak ketiga sampai pekerjaan selesai dan diverifikasi oleh pemberi kerja. Pekerja dijamin dibayar, pemberi kerja dijamin pekerjaan diselesaikan.

**Pelajari lebih lanjut:** [Sistem Dana Jaminan — Dokumentasi Lengkap](/docs/fitur/sistem-dana-jaminan)

### Dana Perlindungan

Dana proteksi yang dikelola platform untuk menangani perselisihan atau penipuan. Dana dari pool ini dapat diklaim oleh pihak yang dirugikan setelah proses review oleh tim Daily Worker Hub.

**Pelajari lebih lanjut:** [Dana Perlindungan — Dokumentasi Lengkap](/docs/fitur/dana-perlindungan)

## Fitur Operasional

### Pencarian Lowongan (Pekerja)

| Fitur | Deskripsi |
|-------|-----------|
| **Filter Lokasi** | Pilih area di Bali: Seminyak, Kuta, Canggu, Ubud, Sanur, Jimbaran, Nusa Dua |
| **Filter Jenis Pekerjaan** | F&B, Housekeeping, Receptionist, Driver, Kitchen, Bartender, Special Events |
| **Filter Rate Harian** | Range minimal dan maksimal bayaran per hari |
| **Filter Jadwal** | Weekdays, weekends, atau tanggal spesifik |
| **Sortir** | Terdekat, Bayaran Tertinggi, Terbaru |
| **Bookmark** | Simpan lowongan menarik untuk apply nanti |
| **Notifikasi Lowongan Baru** | Dapatkan alert saat lowongan baru posting sesuai preferensi Anda |

### Posting Lowongan (Pemberi Kerja)

| Fitur | Deskripsi |
|-------|-----------|
| **Judul & Kategori** | Deskripsi jelas posisi yang dibutuhkan |
| **Detail Tugas** | Penjelasan lengkap tugas dan tanggung jawab |
| **Lokasi & Peta** | alamat dan koordinat untuk referensi pekerja |
| **Jadwal** | Tanggal, jam masuk, jam pulang, durasi shift |
| **Rate Harian** | Besaran bayaran dan apakah termasuk meal/transport |
| **Persyaratan** | Sertifikat, pengalaman, kemampuan khusus |
| **Foto/Tempat Usaha** | Visual tempat kerja untuk menarik pekerja |
| **Auto-Closing** | Lowonganauto-close setelah jumlah pelamar terpenuhi |

### Chat Real-Time

| Fitur | Deskripsi |
|-------|-----------|
| **Messaging Langsung** | Pekerja dan pemberi kerja dapat chat tanpa perantara |
| **Kirim Foto/Dokumen** | Kirim foto tempat meeting atau dokumen tambahan |
| **Typing Indicator** | Ketahuilah saat lawan chat sedang mengetik |
| **Read Receipts** | Konfirmasi pesan sudah dibaca |
| **History Chat** | Semua percakapan tersimpan dan dapat diakses kapan saja |

### Sistem Rating & Review

| Fitur | Deskripsi |
|-------|-----------|
| **Rating 1-5 Bintang** | Pemberi Kerja menilai pekerja setelah pekerjaan selesai |
| **Review Tertulis** | Ulasan dalam bentuk teks sebagai referensi |
| **Public Profile Rating** | Rating terlihat di profil pekerja untuk pemberi kerja lain |
| **Pemberi Kerja Rating** | Pekerja juga bisa memberi rating untuk pemberi kerja |

## Benefit untuk Setiap Tipe Pengguna

### Benefit untuk Pekerja

| Benefit | Penjelasan |
|---------|------------|
| **Tanpa Perantara** | Melamar langsung ke pemberi kerja tanpa calo |
| **Pembayaran Terjamin** | Dana dihold Dana Jaminan — tidak ada risk tidak dibayar jika pekerjaan selesai |
| **Fleksibilitas Penuh** | Pilih sendiri berdasarkan jadwal, lokasi, rate, dan jenis pekerjaan |
| **Proteksi Sengketa** | Dana Perlindungan melindungi Anda jika terjadi perselisihan |
| **Build Reputasi** | Rating dan review meningkatkan peluang diterima di lowongan berikutnya |
| **Komunitas Hospitality Bali** | Akses ke network bisnis hospitality besar di Bali |
| **Bayaran Cepat** | Setelah approval dari pemberi kerja, dana langsung masuk dan bisa dicairkan |

### Benefit untuk Pemberi Kerja

| Benefit | Penjelasan |
|---------|------------|
| **Tenaga Kerja Terverifikasi** | Semua pekerja telah diverifikasi identitasnya |
| **Dana Aman** | Deposit di Dana Jaminan sampai pekerjaan selesai dan sesuai |
| **Hemat Biaya** | Tidak ada komisi untuk calo atau agency |
| **Proses Cepat** | Posting hingga hiring selesai dalam hitungan jam |
| **Proteksi Sengketa** | Dana Perlindungan menangani perselisihan secara adil |
| **Rating & Screening** | Review dari pemberi kerja lain membantu Anda memilih pekerja terbaik |
| **Fleksibilitas Jadwal** | Hire pekerja untuk shift harian, mingguan, atau proyek tertentu |
| **Tidak ada Hidden Cost** | Semua biaya transparan di muka |

## Mengapa Fitur Ini Penting?

| Masalah Umum | Solusi Daily Worker Hub |
|--------------|----------------------|
| Pekerja tidak dibayar setelah selesai kerja | Dana Jaminan menahan dana sampai diverifikasi |
| Employercurang tidak mau approve pembayaran | Dana Perlindungan menangani perselisihan |
| Calo mengambil komisi besar | Direct hiring tanpa perantara |
| Pekerja tidak kredibel (KTP palsu) | Verifikasi identitas wajib untuk semua |
| Pemberi Kerja tidak dikenal dan sulit dipercaya | Rating & review membangun kepercayaan |
| Pembayaran terlambat atau tidak jelas | Sistem Dana Jaminan memastikan pembayaran tepat waktu |
| Sengketa tanpa penyelesaian | Dana Perlindungan + tim review yang adil |
