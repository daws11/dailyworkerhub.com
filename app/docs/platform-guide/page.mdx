# Platform Guide

Cara menggunakan platform DailyWorkerHub secara efektif.

## Untuk Daily Workers

Jika Anda mencari pekerjaan harian, panduan ini akan membantu Anda:
- [Cara Mencari Lowongan](/docs/platform-guide/cara-mencari-lowongan) - Tips menemukan lowongan yang sesuai

## Untuk Pemberi Kerja

Jika Anda perlu menyewa daily workers:
- [Cara Posting Lowongan](/docs/platform-guide/cara-posting-lowongan) - Langkah memposting lowongan pekerjaan

## Fitur Utama

- **Search** - Cari lowongan berdasarkan lokasi, kategori, dan harga
- **Filter** - Saring hasil berdasarkan preferensi Anda
- **Booking** - Pesan worker langsung melalui platform
- **Payment** - Pembayaran aman melalui Escrow System
- **Protection** - Protection Pool untuk keamanan transaksi
