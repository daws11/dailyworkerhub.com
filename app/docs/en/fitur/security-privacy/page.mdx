# Security & Privacy

## Data Protection

Daily Worker Hub uses industry-standard security measures to protect your data:

### Encryption
- **TLS 1.3** for all data in transit
- **AES-256** encryption for sensitive data at rest
- **bcrypt** hashing for passwords

### Access Control
- **Row Level Security (RLS)** — You can only access your own data
- **Two-Factor Authentication (2FA)** available via WhatsApp
- **Session management** with automatic timeouts

### Compliance
- Compliant with Indonesia's Personal Data Protection Law (UU PDP)
- Regular security audits and penetration testing
- Transparent data retention policies
