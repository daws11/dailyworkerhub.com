# Basic Preparation

## What You Need to Get Started

Before creating your account on Daily Worker Hub, make sure you have the following ready:

### Identity Documents
- **KTP** (Indonesian ID card) or **Passport** (for foreign nationals)
- A clear photo of your face for verification
- Active phone number (WhatsApp preferred)

### Bank Account
- Active bank account in your name for receiving payments
- Supported banks: BCA, Mandiri, BNI, BRI, and major Indonesian banks

### Device Requirements
- Smartphone with camera (for QR code check-in)
- Internet connection
- Latest version of Chrome, Safari, or Firefox browser

## Next Steps

Once you have everything ready, proceed to [Create an Account](/docs/en/getting-started/membuat-akun).
