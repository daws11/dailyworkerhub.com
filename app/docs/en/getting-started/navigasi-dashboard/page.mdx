# Dashboard Navigation

The dashboard is the main control center in Daily Worker Hub. From here, you can access all platform features — from job search, application management, chat with employers, to fund disbursement. This page explains each dashboard section in detail.

---

## Worker Dashboard View

If you registered as a **worker**, here is the layout and function of each dashboard section:

### Worker Menu Structure

| Position | Menu |
|--------|------|
| Left (Logo) | Daily Worker Hub Home |
| Navigation | **Search Jobs** · **My Applications** · **Balance** · **Profile** |

### Main Menus

| Menu | Function | Shortcut |
|------|--------|---------|
| **Home** | Overview: latest job listings, notifications, statistics | Click Daily Worker Hub logo |
| **Search Jobs** | Job search and filters | — |
| **My Applications** | List of applied jobs and their statuses | — |
| **Chat** | Conversations with employers | — |
| **My Balance** | Platform balance, transaction history, withdraw funds | — |
| **Profile** | Edit profile, verification, account settings | — |

### Home (Worker Dashboard Home)

**Displayed content:**

| Section | Description |
|--------|-----------|
| **Latest Notifications** | Alerts for accepted applications, messages from employers, platform updates |
| **Jobs Matching Preferences** | Job recommendations based on profile and location |
| **Latest Job Listings** | List of the 10 newest job listings on the platform |
| **My Statistics** | Total applications, active applications, total earnings this month |
| **Promo Banner** | Info on premium features, events, or platform updates |

### Search Jobs Menu

**Available features:**

| Feature | Description |
|-------|-----------|
| **Search Bar** | Type keywords for quick search |
| **Location Filter** | Select area: Seminyak, Kuta, Canggu, Ubud, etc. |
| **Category Filter** | Select job type: Waiter, Kitchen, Housekeeping, etc. |
| **Rate Filter** | Enter minimum and maximum daily rate range |
| **Schedule Filter** | Select work days: weekdays, weekends, specific dates |
| **Sort** | Sort by: Newest, Nearest, Highest Pay |
| **Bookmark** | Save interesting job listings to apply later |
| **Apply** | Apply directly with one click |

### My Applications Menu

**Page structure:**

| Tab | Displays |
|-----|-------------|
| **All** | All jobs you have ever applied to |
| **Pending** | Applications that have not yet received a response from the employer |
| **Accepted** | Applications that have been accepted — proceed to hiring stage |
| **Rejected** | Applications rejected by the employer |
| **Expired** | Job listing was closed before you were accepted |

**Details per application:**

| Information | Description |
|----------|-----------|
| **Employer Name** | Business that posted the job listing |
| **Position** | Title you applied for |
| **Location** | Work area |
| **Rate** | Daily pay offered |
| **Status** | Pending / Accepted / Rejected / Expired |
| **Application Date** | When you submitted the application |
| **Action** | View details, chat employer, cancel application |

### My Balance Menu

**Page structure:**

| Section | Description |
|--------|-----------|
| **Available Balance** | Amount currently withdrawable |
| **Pending Balance** | Funds in process (not yet released) |
| **Total Earnings** | All earnings since joining |
| **Earnings This Month** | Total earnings in the current month |
| **Withdraw Button** | Action to disburse balance to bank account |

**Transaction History:**

| Column | Description |
|-------|-----------|
| **Date** | When transaction occurred |
| **Type** | Release from Security Deposit / Withdrawal / Bonus |
| **Amount** | Transaction amount |
| **From** | Employer who paid (for releases) |
| **Status** | Completed / Pending / Failed |

### Profile Menu

**Available sub-menus:**

| Sub-menu | Function |
|----------|--------|
| **Edit Profile** | Change photo, name, description, skills |
| **Account Verification** | View verification status, complete pending ones |
| **Portfolio** | Upload work photos/videos (optional) |
| **Rating & Reviews** | View reviews from employers |
| **Settings** | Change password, enable 2FA, manage notifications |
| **Logout** | Log out of account |

## Employer Dashboard View

If you registered as an **employer**, here is the layout and function of each dashboard section:

### Employer Menu Structure

| Position | Menu |
|--------|------|
| Left (Logo) | Daily Worker Hub Home |
| Navigation | **Post Job** · **My Jobs** · **Balance** · **Profile** |

### Main Menus

| Menu | Function |
|------|--------|
| **Home** | Overview: active job listings, latest applicants, hiring statistics |
| **Post Job** | Create and publish new job listings |
| **My Jobs** | Manage all posted job listings |
| **Worker Database** | Search and view registered worker profiles |
| **Chat** | Conversations with applicants |
| **Balance** | Deposit, view Security Deposit transaction history |
| **Profile** | Edit business profile, verification, settings |

### Home (Employer Dashboard Home)

**Displayed content:**

| Section | Description |
|--------|-----------|
| **Latest Notifications** | Alerts for new applicants, messages from workers, platform updates |
| **Active Job Listings** | Number of currently open job listings |
| **New Applicants** | List of workers who applied today |
| **Pending Transactions** | Hiring currently in process |
| **Hiring Statistics** | Total workers hired, total spending this month |
| **Quick Actions** | Quick buttons: Post Job, Deposit Balance |

### Post Job Menu

**Form fields to fill:**

| Field | Required? | Description |
|-------|--------|-----------|
| **Title** | ✅ | Position needed |
| **Category** | ✅ | Type of work |
| **Location** | ✅ | Address or area |
| **Date** | ✅ | Date and time of work |
| **Rate** | ✅ | Daily pay |
| **Description** | ✅ | Detailed tasks and requirements |
| **Photo** | Optional | Workplace images |
| **Requirements** | Optional | Certificates, experience, etc. |

### My Jobs Menu

**Page tabs:**

| Tab | Displays |
|-----|-------------|
| **All** | All job listings ever posted |
| **Active** | Job listings currently open |
| **Draft** | Job listings not yet published |
| **Hired** | Active hiring |
| **Completed** | Job listings that have finished |

**Actions per job listing:**

| Action | Function |
|------|--------|
| **View Applicants** | View and select workers who applied |
| **Edit** | Change job listing details (if no one has been hired yet) |
| **Close** | Close job listing early |
| **Repost** | Create a duplicate for reposting |
| **Delete** | Delete draft job listing |

### Balance Menu

**Sub-menus:**

| Sub-menu | Function |
|----------|--------|
| **Balance Summary** | Total available Security Deposit balance |
| **Deposit** | Top up Security Deposit balance via bank transfer / e-wallet |
| **Deposit History** | Log of all deposits made |
| **Active Transactions** | Ongoing hiring |
| **Transaction History** | All completed Security Deposit transactions |

## Important UI Elements

### Icons and Indicators

| Icon | Meaning |
|------|---------|
| 🔔 | Notification — new message or update |
| ✅ | Completed / Passed / Approved |
| ❌ | Rejected / Failed / Cancelled |
| ⏳ | Pending / In process |
| ⭐ | Rating / Review |
| 💰 | Balance / Financial transaction |
| 📍 | Location |
| 📅 | Date / Schedule |
| ⭐ (badge) | Premium / Verified Worker |

### Application and Job Listing Statuses

| Status | Color | Meaning |
|--------|-------|------|
| **Active / Open** | Green | Job listing is still accepting applicants |
| **Pending** | Yellow | Waiting for response from other party |
| **Accepted / Hired** | Blue | Application accepted, hiring process begins |
| **Rejected** | Red | Rejected or not suitable |
| **Completed** | Gray | Execution completed |
| **Expired / Closed** | Gray | No longer active |
| **Cancelled** | Red | Cancelled by one party |
| **Disputed** | Orange | A dispute is being processed |

## Navigation Tips

| Tip | Explanation |
|------|------------|
| **Use search** | To find workers or job listings quickly |
| **Enable push notifications** | So you don't miss applications or new applicants |
| **Filter as needed** | Narrow down search results for more relevant results |
| **Bookmark interesting job listings** | To apply later without having to search again |
| **Check dashboard regularly** | Especially in the morning for same-day job listings |
| **Update profile periodically** | Increases visibility and chances of being hired |

---

## Navigation Troubleshooting

| Problem | Solution |
|---------|--------|
| **Menu not appearing** | Refresh the page or clear browser cache |
| **Error when loading data** | Check internet connection, try again in a few minutes |
| **Notifications not received** | Enable notifications in browser/device settings |
| **Blank page** | Logout and login again, or use a different browser |
| **Data not updating** | Pull-to-refresh on mobile, or click refresh on desktop |
