## What is Daily Worker Hub?

Daily Worker Hub is a platform connecting daily freelance workers with businesses across Indonesia. We eliminate middlemen and create a transparent, fair work ecosystem for hospitality, cleaning, events, and more.

### For Workers

- Find daily jobs without intermediaries
- Build your reputation with verified work history
- Get paid directly after each shift via our escrow system
- Community protection fund for emergencies

### For Businesses

- Recruit trusted daily workers with verified track records
- Transparent payment system — pay only for completed work
- QR code attendance tracking
- Built-in dispute resolution

## Platform Status

Currently serving **10,000+ verified workers** and **500+ active businesses** across Indonesia — all without middlemen.
